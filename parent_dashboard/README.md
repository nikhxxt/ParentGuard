# Parent Dashboard - ParentGuard

React Web App to monitor child devices.

## Features
- Real-time notifications
- Location tracking
- Audit logs
- Consent management
- Cross-platform: Android full features, iPhone supervised features
- Responsive UI (mobile + desktop)
